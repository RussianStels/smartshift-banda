# 📦 Инструкция по установке Banda Schedule Helper

## Шаг 1: Создайте папку для расширения

Создайте папку, например: `banda-extension`

## Шаг 2: Скачайте все файлы

Сохраните в эту папку следующие файлы:

1. **manifest.json** (конфигурация)
2. **background.js** (фоновый процесс)
3. **content.js** (перехват данных)
4. **popup.html** (интерфейс)
5. **popup.js** (логика)

## Шаг 3: Создайте иконку

### Вариант А: Автоматически (быстро)

1. Сохраните файл **icon.html** в ту же папку
2. Откройте **icon.html** в браузере
3. Иконка **icon.png** скачается автоматически
4. Переместите **icon.png** в папку расширения

### Вариант Б: Вручную

Создайте любое изображение 128x128 пикселей и сохраните как **icon.png**

Или используйте эмодзи:
- Перейдите на https://www.emojiall.com/en/emoji/📅
- Скачайте изображение как **icon.png**

## Шаг 4: Структура папки

После всех шагов в папке должно быть:

```
banda-extension/
  ├── manifest.json
  ├── background.js
  ├── content.js
  ├── popup.html
  ├── popup.js
  └── icon.png
```

## Шаг 5: Загрузите расширение в Chrome

1. Откройте Chrome
2. Перейдите по адресу: `chrome://extensions/`
3. Включите **"Режим разработчика"** (переключатель в правом верхнем углу)
4. Нажмите **"Загрузить распакованное расширение"**
5. Выберите папку **banda-extension**
6. Готово! ✅

## Шаг 6: Первый запуск

1. Откройте https://banda.tseh85.ru
2. Авторизуйтесь
3. Перейдите в раздел **Табель** (https://banda.tseh85.ru/table)
4. Откройте DevTools (F12) → вкладка **Console**
5. Обновите страницу (F5)
6. В консоли должно появиться:
   ```
   ✅ Banda Extension: Content script loaded
   🔑 Token intercepted: eyJhbGc...
   🆔 IDs found: { employeeId: '...', departmentId: '...' }
   ```

7. Теперь кликните на иконку расширения в панели Chrome
8. Расширение загрузит ваше расписание! 🎉

## ⚠️ Если расширение не работает

### Проблема: "Manifest file is missing or unreadable"

**Решение**: Проверьте, что файл **manifest.json** сохранен БЕЗ расширения .txt

- Windows: В проводнике включите "Расширения имен файлов"
- Убедитесь, что файл называется `manifest.json`, а не `manifest.json.txt`

### Проблема: "Токен не найден"

**Решение**: 

1. Откройте DevTools (F12)
2. Вкладка **Network**
3. Фильтр: XHR
4. Обновите страницу табеля
5. Найдите запрос к `api.banda.tseh85.ru`
6. Во вкладке **Headers** скопируйте значение `Authorization: Bearer ...`
7. Откройте расширение → DevTools расширения (правый клик → Inspect)
8. В консоли выполните:
   ```javascript
   chrome.storage.local.set({ bearerToken: 'ВАШ_ТОКЕН_БЕЗ_BEARER' })
   ```

### Проблема: "ID не найдены"

**Решение**:

1. На странице табеля откройте URL
2. Скопируйте `employeeId` и `departmentId` из адресной строки или из Network запросов
3. В консоли расширения выполните:
   ```javascript
   chrome.storage.local.set({ 
     employeeId: 'ВАШ_EMPLOYEE_ID',
     departmentId: 'ВАШ_DEPARTMENT_ID'
   })
   ```
